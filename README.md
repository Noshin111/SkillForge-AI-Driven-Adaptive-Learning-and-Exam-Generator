# SkillForge  
A web-based skill development and assessment platform.

##  Overview
SkillForge is a learning platform designed to help users improve job-ready skills through structured courses and timed assessments.  
It includes authentication, a dashboard, online exams, and performance tracking.

##  Features
- User Registration & Login
- Secure Authentication
- Course Dashboard
- Timed MCQ-Based Exams
- Automatic Result Calculation
- Performance Overview

##  Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express.js  
Database: MongoDB  
Authentication: JWT, bcrypt
